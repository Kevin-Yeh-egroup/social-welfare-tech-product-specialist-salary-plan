# 社福科技產品專員薪資落點與入職規劃報告

Public noindex review artifact for a de-identified salary positioning and onboarding plan.

## Public Boundary

- This repository is public and link-accessible.
- Search indexing is blocked with HTML meta robots, `robots.txt`, and `X-Robots-Tag`.
- Noindex is not privacy protection.
- Do not add names, phone numbers, email addresses, private document links, internal local paths, bank data, ID numbers, private notes, or unredacted HR records.
- Salary numbers are retained because the artifact is a salary-positioning case report.

## Files

- `index.html`: public-safe report page.
- `robots.txt`: review-stage crawler block.
- `vercel.json`: noindex response header.
- `reports/public-redaction-scan-report.json`: local scan result before publication.
- `PUBLIC_PRODUCTION_LOG.md`: deployment proof and verification notes.

## Review Rule

Actual hiring, termination, salary, offer, and probation decisions must stay with a human reviewer. This page is a planning artifact, not an automated HR decision system.
